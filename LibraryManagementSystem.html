Library Management System

Tools & Technologies Used


  • Java – Core application logic (Object-Oriented Programming).
  • Java Swing / JavaFX – GUI design (Desktop app).
  • MySQL – Database for storing book/member/issue records.
  • JDBC – Connecting Java to MySQL.
  • NetBeans / IntelliJ IDEA – IDE for development.
  • MySQL Workbench – Database management.
  • Git & GitHub – Version control and project hosting.
  • Figma – UI/UX design mockups
  • Draw.io – ER diagrams and system architecture.
  • Document Object Model (DOM) – For modern GUI development, consider using JavaFX, which provides a more flexible and modern event-handling mechanism.

Key Features


  • Book Management – Add, update, delete, and search books
  • Member Registration – Manage student/member records
  • Issue & Return System – Track borrowed and returned books
  • Due Date Management – Automatically calculate return dates
  • Fine Calculation – Auto-fine for late returns
  • Search Functionality – Search books/members by ID, name, etc.
  • Dashboard – Summary of total books, members, issues, etc.
  • Report Generation – Daily issue logs and export options
  • Admin Login – Simple authentication for admin access

Project Overview

Library Management System
A desktop application built using Java and MySQL to manage books, members, and borrow/return workflows for libraries.

Tools: Java,Swing,MySQL,JDBC,IntelliJ IDEA,Git
Features: Book issue,book return,member management,fine calculation,admin login,real-time search,and database integration.
