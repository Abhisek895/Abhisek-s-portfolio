Simon Game

Key Features

  • Dynamic Sequence Generation – Each level increases in complexity with an added random color.
  • Real-Time Input Validation – Immediate feedback on correct or incorrect button presses.
  • Sound Integration – Distinct tones for each color using Web Audio API or preloaded audio files.
  • Game Over Detection – Displays current level and progression.
  • Responsive UI – Works across devices with touch and click input compatibility.
  • Fast Restart – Easy restart after game over with a single keypress.
  • Color Highlighting – Colors are visually emphasized when selected to track moves.
  • Touch Support – Fully compatible with touch input for mobile and tablet users.

Tech Stack

  • HTML – Markup structure for game elements (buttons, layout)
  • CSS – Styling, animations, and responsive layout.

  • JavaScript (Vanilla JS )- Core game logic, event handling, and user interaction
  • DOM Manipulation (Vanilla JS )- Real-time updates based on player actions

Project Overview

  • DOM Manipulation – Handling user interactions and updating the UI dynamically
  • Event Handling – Capturing clicks and managing game logic in real-time
  • Responsive Design – Layout adapts across devices using CSS
  • Randomization Logic – Generating unpredictable color sequences for gameplay
  • Audio Feedback – Enhancing interactivity with sound using JavaScript
  • Game Loop Logic – Managing game rounds, checking user input, and resetting the game
